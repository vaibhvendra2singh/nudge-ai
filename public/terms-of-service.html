2. Description of Service

Nudge is a browser-based, AI-powered task management application designed to help individuals organize, prioritize, and act on tasks before deadlines are missed. The Service includes:

  • Task creation, editing, and deletion with deadline tracking
  • Automatic urgency classification (Urgent / Soon / Future) based on real-time deadline proximity
  • AI-generated task breakdowns (subtasks) powered by Google Gemini
  • AI-generated motivational nudge messages for urgent tasks
  • Voice-to-task input using the browser's Web Speech API combined with AI extraction
  • Calendar event export in standard .ics format, compatible with Google Calendar, Apple Calendar, and Outlook
  • Optional Google Calendar integration for scheduling conflict detection
  • Offline functionality via service worker caching
  • Task pattern analytics and AI-driven predictive risk flagging

The Service is provided free of charge. No account registration is required to use Nudge.
